Program do obliczania ceny produktu dla klienta. Użytkownik wpisuje bazową wartość przedmiotu, a następnie pożądaną marżę.

Sposób kompilacji:
1. Uruchom windows Powershell
2. Korzystając z komendy <cd .\nazwafolderu1\nazwafolder2\...\folder_z_plikiem>
dojdź do folderu, gdzie znajduje sie plik src.c
3. Użyj komendy <gcc -Wall -o Program.exe src.c>, by skompilowac Program
4. Uruchom program używając komendy <.\Program.exe>
5. Enjoy

Sposób użytkowania:
1. Wpisz podstawową cenę produktu i wciśnij enter
2. Wpisz przewidywaną marżę i wciśnij enter
3. Program wyświetli obliczoną cenę dla klienta
