# Różnica bezwzględna
Program szukajacy najwiekszej bezwzględnej różnicy między dwoma sąsiadującymi liczbami. Użytkownik najpierw wpisuje jak dużo liczb będzie wpisywał (>2). Następnie użytkownik wypisuje każdą liczbę po kolei. Po wykonaniu tego program wykonuje operacje i wypisuje wynik.

## Sposób kompilacji:
1. Uruchom windows Powershell
2. Korzystając z komendy <cd .\nazwafolderu1\nazwafolder2\...\folder_z_plikiem>
dojdź do folderu, gdzie znajduje się plik main.c (komendy zatwierdza się klawiszem **Enter**)
3. Użyj komendy <gcc -Wall - Wextra -o Program.exe main.c>, by skompilowac Program
4. Uruchom program używając komendy <.\Program.exe>
5. Enjoy

## Sposób użytkowania:
1. Wypisz liczbę (całkowitą, większą od 2) i kliknij **Enter** - tyle liczb zostanie porównanych przez program
2. Wypisz liczbę (całkowitą), którą chcesz porównać i kliknij **Enter**. Ten krok powtarzasz dopóki nie wpiszesz tyle liczb, by ich suma wynosiła tyle co liczba wpisana krok wcześniej
3. Po wpisaniu wszystkich liczb, program wypisuje wynik, czyli największą różnicę bezwzględną między dwoma sąsiadującymi liczbami
